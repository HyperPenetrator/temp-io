### **Quick Deploy (2 Minutes)**

1. **Push to GitHub:**
   ```bash
   cd d:\Projects\Weather
   git init
   git add .
   git commit -m "Industry-grade Temp.io by HyperPenetrator"
   
   # Create repo: https://github.com/new
   # Name: temp-io
   
   git remote add origin https://github.com/HyperPenetrator/temp-io.git
   git branch -M main
   git push -u origin main
   ```

2. **Deploy Frontend (Vercel):**
   - Click "Deploy to Vercel" button in README
   - Or visit: https://vercel.com/new
   - Import `temp-io` repository
   - Root: `frontend`
   - Framework: Vite
   - Deploy!

3. **Deploy Backend (Railway):**
   - Click "Deploy to Railway" button in README
   - Or visit: https://railway.app/new
   - Import `temp-io` repository
   - Auto-detects Python
   - Deploy!

4. **Connect Them:**
   - Copy Railway backend URL
   - Add to Vercel: `VITE_API_URL` = Railway URL
   - Redeploy Vercel

5. **Done!** ğŸ‰
   - Your app is live at: `https://temp-io.vercel.app`
